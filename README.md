# Automotive Sales Performance Dashboard

This project was developed as part of the **IBM Data Visualization with Python** certification. It features an interactive web dashboard designed to analyze historical automotive sales data, specifically focusing on the impact of economic recession periods on different vehicle types.

## 🚀 Project Overview
The application provides a dynamic interface to explore sales trends and market shares. Users can toggle between:
1. **Recession Period Statistics:** Analyzing how different car types performed during historical economic downturns.
2. **Yearly Statistics:** A detailed breakdown of sales performance for a specific selected year.

## Tech Stack
- **Python 3.8**: Core programming language.
- **Dash**: Framework for building the web analytical application.
- **Plotly**: For creating interactive, high-quality graphs (Line, Bar, and Pie charts).
- **Pandas**: Data manipulation, cleaning, and aggregation.
- **Seaborn/Matplotlib**: Used for initial exploratory data analysis (EDA).

## Key Features
- **Interactive Callbacks**: The dashboard automatically enables or disables the year selection dropdown based on the chosen report type (UX best practice).
- **Multiple Visualizations**: 
    - Average sales trends over time.
    - Sales fluctuations by vehicle type.
    - Expenditure maps and market share distributions.
    - Correlation between advertisement expenditure and sales volume.
